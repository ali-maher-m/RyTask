<ComingSoon tier="MVP scope" />

RyTask plans custom fields so you can attach your own data to work items — without the
caps and paywalls common elsewhere (ClickUp limits uses, Plane and Asana gate the
feature). The first wave covers the core types; richer types and rules follow.

What's planned:

- **Core field types** — text, number, select, date, user (MVP scope).
- **All field types** — the full set beyond the core five (v2).
- **Field-level required and validation rules** — enforce data quality (v2).
- **Workflow transition rules** — restrict who can move items, and when (v3).
- **Formula and rollup fields** — computed values across items (v3).
