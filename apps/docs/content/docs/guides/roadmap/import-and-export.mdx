<ComingSoon tier="MVP scope" />

RyTask plans to make data movement boring, in the best way. Getting data in: CSV import
and, later, importers from the tools people leave. Getting data out: a **full export in
an open format** — not the crippled partial exports SaaS tools use to keep you in.
Self-hosting only counts as owning your data if you can actually take all of it with you.

What's planned:

- **CSV import and export** — bulk in and out (MVP scope).
- **Full data export in an open format** — a complete dump of everything (MVP scope).
- **API-based bulk export** — programmatic backup without punishing rate limits (MVP scope).
- **Importers from Jira, Linear, Trello, and Asana** — migration without friction (v2).
- **Scheduled backups** — automated dumps for self-hosted instances (v2).
