This section has one page per planned capability cluster from the RyTask feature matrix.
**Nothing in this section is shipped.** Every page describes intent, not behavior — there
are no screenshots because there is nothing to screenshot yet, and details may change
before they ship.

Each page carries a banner with its tier: **MVP scope** means it belongs to the Stage 1
feature set still being built, **v2** and **v3** are later waves. Where a cluster mixes
tiers, the banner reflects the main capability and the bullet list marks the rest.

For a single table of what is live, in progress, and planned, see the
[feature status reference](/docs/reference/feature-status).
