title: How-to guides
description: Task-focused guides for hosting, administering, and using RyTask.
---

Each guide gets one job done. They assume RyTask is already running — if it isn't, start with
the [quickstart](/docs/tutorials/quickstart).

- **[Self-hosting & administration](/docs/guides/self-hosting)** — requirements, production
  setup, environment variables, TLS, backups, upgrades, and scaling.
- **[Using RyTask](/docs/guides/users)** — one page per feature area, written so a teammate who
  never opens a terminal can follow along.
- **[Slack integration](/docs/guides/slack)** — install the bot, capture with `/task`, map
  users, troubleshoot.
- **[MCP & AI agents](/docs/guides/mcp)** — connect Claude, Claude Code, or any MCP client, and
  understand the security model.
- **[Other integrations](/docs/guides/integrations)** — GitHub and friends (coming soon).
